O que é WebPack

Um 'Module Bundler' - Desenvolveremos nossa aplicação utilizando um sistema de modulos
- Ver como funciona o sistema Common JS
- Ver como funciona o sistema de módulos do ES6 (2015)
- Gerar arquivo final que será usado no HTML, em ES5
- Subir servidor para Desenvolvimento
- Utilizar React Hot Loader para agilizar o desenvolvimento

Objetivo do WebPack

Pegar os módulos que temos juntar em um pacote(Bundle) para entregar para o navegador um Bundle com o ES5 formato que o browser consegue fazer leitura
